# GoWinDo it — нейро-агентство полного цикла

Лендинг-страница для нейро-агентства GoWinDo it.

## Стек

- HTML / CSS / JS (vanilla, без фреймворков)
- Шрифты: Outfit (display) + Inter (body) + JetBrains Mono (mono/code)
- Деплой: Vercel

## Локальная разработка

Открыть `index.html` в браузере — всё работает без сервера.
